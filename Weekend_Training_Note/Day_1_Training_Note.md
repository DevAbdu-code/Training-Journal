#  Introduction

The first day of training focused on the basic concepts of security and cybersecurity, followed by an introduction to Linux and virtualization. These concepts form the foundation for understanding modern cybersecurity practices.

---

#  What is Security?

Security is the practice of protecting systems, networks, and data from unauthorized access, damage, or theft.

---

#  Types of Security

- Physical Security (protecting hardware and buildings)  
- Network Security (protecting network systems)  
- Information Security (protecting data)  
- Cybersecurity (protecting digital systems)  

---

#  What is Cybersecurity?

Cybersecurity is the process of protecting computers, networks, and data from cyber attacks, unauthorized access, and damage.

---

#  Importance of Cybersecurity

- Protects sensitive information  
- Prevents cyber attacks  
- Ensures privacy and safety  
- Maintains system integrity  
- Supports national and organizational security  

---

#  Classification of Cybersecurity

##  Offensive Security (Red Team)
- Focus: Attacking systems to find vulnerabilities  
- Activities:
  - Penetration testing (Pentesting)  
  - Ethical hacking  
  - Vulnerability assessment  

---

##  Defensive Security (Blue Team)
- Focus: Protecting systems from attacks  
- Activities:
  - Monitoring systems  
  - Incident response  
  - Security analysis  

---

#  Domains of Security

Security can be classified based on domains:

- Land  
- Sea  
- Space  
- Cyber (Cybersecurity)  

---

#  What is Hacking?

Hacking is the process of finding and exploiting weaknesses in a system.

---

#  Types of Hackers

- White Hat (ethical hackers)  
- Black Hat (malicious hackers)  
- Gray Hat (in between)  

---

#  Introduction to Linux

Linux is an open-source operating system widely used in cybersecurity.

---

#  Why Linux is Different from Other Operating Systems

- Open-source (free and customizable)  
- More secure and stable  
- Powerful command-line interface  
- Widely used in servers and cybersecurity  

---

#  Why Linux is Used in Cybersecurity

- Contains many security tools  
- Supports scripting and automation  
- Gives more control to users  
- Preferred by ethical hackers and professionals  

---

# What is Virtualization?

Virtualization is the process of creating a virtual version of a computer system inside another system.

👉 It allows running multiple operating systems on one physical machine.

#  Why We Use VirtualBox

- Beginner-friendly  
- Easy to install and use  
- Allows running Linux inside Windows  
- Safe environment for practice
